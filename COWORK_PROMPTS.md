# Coworkプロンプト集（初心者向け）

このファイルは、ClaudeのCowork機能でそのまま使える定型文です。
必要なものをコピペして使ってください。

## 1) 実装前の整理

```text
目的: POST /api/echo を追加したいです。
初心者向けに、まず実装前の方針を3行で説明してください。
その後、最小手順で実装してください。
```

## 2) 小さく実装して確認

```text
一気に作らず、1ステップずつ進めたいです。
まず正常系だけ実装して、確認方法を先に示してください。
確認が終わったら次のステップに進みます。
```

## 3) エラー対応

```text
このエラーの原因候補を3つ挙げてください。
それぞれの確認コマンドと、初心者向けの対処手順を示してください。
```

## 4) Gitコミット補助

```text
今の変更を見て、コミットを何個に分けるべきか提案してください。
各コミットのメッセージ案も出してください。
```

## 5) 学習モード

```text
この変更について、なぜその実装にしたかを初心者向けに説明してください。
難しい用語には1行の補足をつけてください。
```

## 6) レビュー依頼

```text
この変更をレビューしてください。
優先順で「危険な点」「壊れる可能性」「改善提案」を短く示してください。
```
