# CLAUDE.md

このファイルは、Claude Code がこのプロジェクトで作業するときのガイドです。

## プロジェクト概要

- 種類: Node.js + Express の練習用 API
- 目的: 初心者が API 開発と Git 運用に慣れること
- メインファイル: `src/index.js`

## 回答スタイル

- 返答は日本語で行う
- 専門用語は短く説明を添える
- 手順は「今やること」を先に示す
- 可能ならコマンドはコピペ可能な形で提示する

## 実装ルール

- 変更は最小単位で行う
- 1回の変更ごとに動作確認方法を示す
- 破壊的な操作（削除、リセット）は明示的な依頼があるときのみ行う
- 機密情報（パスワード、トークン、秘密鍵）はコードやコミットに含めない

## API 実装方針

- 新しい API は `src/index.js` に追加する
- まず `GET` の簡単なエンドポイントから作る
- レスポンスは JSON で返す
- エラー時も JSON で返す（将来的に統一する）

## 開発時の基本コマンド

- 依存関係インストール: `npm install`
- 開発起動: `npm run dev`
- 通常起動: `npm start`
- 変更確認: `git status`

## 優先タスク（次の学習ステップ）

1. `POST /api/echo` を追加する（受け取った JSON を返す）
2. 入力チェックの基本を入れる（必須項目の確認）
3. API 仕様メモを README に追記する
